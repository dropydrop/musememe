# MuseMeme

Générateur de mèmes français avec IA.

Application React + Vite — édition visuelle de mèmes avec rendu canvas temps réel, templates populaires, et génération de légendes par IA.

## Développement

```bash
npm install
npm run dev
```

## Stack

- **React 19** — UI
- **Vite** — bundler dev
- **Tailwind CSS v4** — styles
- **Lucide React** — icônes
- **Gemini API** — génération automatique de légendes (`api/generate-captions.ts`)
